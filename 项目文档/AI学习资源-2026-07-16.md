## 1. 摩根大通 Python Training

- X 来源：[@nuannuan_share，1995026014674833665](https://x.com/nuannuan_share/status/1995026014674833665)
- 官方资源：[JPMorganChase/python-training](https://github.com/jpmorganchase/python-training)
- 定位：为摩根大通业务分析师、交易员和部分客户设计的入门训练，重点是 Python 数值计算与数据可视化，不是完整的计算机科学课程。
- 形式：Jupyter Notebook，仓库同时提供 Binder 运行入口。
- 许可：Apache-2.0。

### 课程映射

适合放入 `1-人工智能概述/动手实践`，作为“为 AI 学习补齐 Python 与数据分析基础”的可选实训。建议优先选 2–3 个 Notebook 做课前体验，不宜把整个仓库包装成完整 Python 培训。

## 2. MedGemma 1.5 与 MedASR

- X 来源：[@aigclink，2011215944530739253](https://x.com/aigclink/status/2011215944530739253)
- 官方资源：[MedGemma 1.5 模型卡](https://huggingface.co/google/medgemma-1.5-4b-it)、[MedASR 模型卡](https://huggingface.co/google/medasr)

### 已核验能力

- MedGemma 1.5 为 4B 多模态指令微调模型，扩展到 CT/MRI 三维体数据、全切片病理图像、多时点胸片、胸片解剖定位、实验室报告结构化提取和 EHR 文本理解。
- MedASR 是面向医疗口述的 105M 参数 Conformer 语音转文字模型。
- X 帖文中“5.2% 词错率”对应官方 Eye Gaze 评测里的 `MedASR + 6-gram language model`；Whisper v3 Large 在同一表中为 12.5%，相对下降约 58%。

### 必须保留的边界

- 更准确的表述是“可下载权重的开放模型”，使用受 Health AI Developer Foundations 条款约束，不宜无条件简化为“完全开源”。
- MedASR 当前为英文模型，对非美式英语、日期格式、新药名和低质量音频可能退化。
- 两个模型都是下游研发起点，输出需要独立核验，不能直接用于诊断、治疗或患者管理决策。

### 课程映射

适合作为“垂直领域多模态模型”案例，用一页说清“能力扩展——本地运行——许可边界——临床验证”，不只展示模型指标。

## 4. NotebookLM 16 个提示词

- X 来源：[@0xluffy_eth，2010549803722408209](https://x.com/0xluffy_eth/status/2010549803722408209)
- 核验结果：书签索引中只有一张图，图片内容为 NotebookLM 标志，没有 16 个提示词；帖文也没有文章正文或其他外部链接。
