# 陈龙彪团队人工智能公开课

一套可直接学习的中文人工智能 Web 课程，共 4 个模块、27 节主题课。每节课同时提供完整讲义、可在“课程精编/原课件”之间切换的 Web PPT、真实图片与视频，以及可下载、可编辑的课程精编 PPTX。

- [查看版权与素材说明](NOTICE.md)

## 公开课件结构

27 份 Web PPT 与 27 个 PPTX 共用同一套结构化内容模型和 336 个 slide ID。每节播放器还关联一份原课件媒体清单，合计呈现 454 张精选 PowerPoint 页、963 个真实图片引用和 24 段可播放视频。公开仓库不包含完整源 PowerPoint、飞书归档或本机路径。

## 本地运行

项目使用 Node.js 22、pnpm 和系统 `unzip`（用于检查 PPTX 包结构）：

```bash
pnpm install --frozen-lockfile
pnpm dev -- --host 127.0.0.1
```

运行结构、课件、哈希、公开边界、语言、内容和生产构建校验：

```bash
pnpm verify
pnpm release:verify
```

校验会确认 4 个模块、27 个课程路由、27 份双模式 Web deck、27 个课程精编 PPTX、336 张精编幻灯片、454 张原课件页、963 个图片引用、24 段可播放视频和 Markdown revision。
