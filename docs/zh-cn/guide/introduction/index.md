# 学习地图

课程按“概念基础 → AI 智能体 → 具身智能 → 行业应用”推进。每一节都是独立学习单元，包含图文讲解、幻灯片、案例、练习、复盘和延伸阅读。

## 第 1 章 · [人工智能概述](/zh-cn/stage-1/)

从人工智能的基本循环、发展历程和机器学习基础进入大模型、生成式 AI 与负责任使用。

1. [人工智能引论：从典型案例到大模型能力边界](/zh-cn/stage-1/ov-001-what-is-ai/) — 从索菲亚、来福机器人和 Crowdverse 等案例出发，依次理解人工智能的定义与历史、社会影响、大模型技术栈、演进分类和能力边界。
2. [人工智能发展与社会影响](/zh-cn/stage-1/ov-002-ai-history-and-society/) — 沿关键技术转折理解人工智能的兴衰，并讨论它如何改变工作、组织和社会分工。
3. [机器学习与深度学习基础](/zh-cn/stage-1/ov-003-machine-learning-and-deep-learning/) — 从数据、模型、损失和泛化进入神经网络、反向传播与卷积网络。
4. [大模型概述、Transformer 与能力边界](/zh-cn/stage-1/ov-004-llm-transformer-and-boundaries/) — 解释 token、注意力、Transformer、预训练与推理，并建立大模型能力边界。
5. [DeepSeek 与开源模型生态](/zh-cn/stage-1/ov-005-deepseek-and-open-models/) — 从推理模型、开源生态和本地部署理解 DeepSeek 带来的技术与应用选择。
6. [提示词、AIGC 与多模态工作流](/zh-cn/stage-1/ov-006-prompt-aigc-and-multimodal/) — 把提示词、文本生成、图像和视频生成组织成可复用、可复核的内容工作流。
7. [幻觉、安全与负责任 AI](/zh-cn/stage-1/ov-007-hallucination-safety-and-responsible-ai/) — 从事实幻觉、提示注入、权限边界和责任链建立可执行的负责任 AI 使用框架。

## 第 2 章 · [AI 智能体](/zh-cn/stage-2/)

从 Agent Loop 进入工具、记忆、规划、多智能体、AI Coding 与可靠性工程。

1. [Agent Loop 与 ReAct](/zh-cn/stage-2/ag-001-agent-loop-and-react/) — 理解智能体如何在观察、推理、行动和反馈之间循环推进任务。
2. [工具调用、MCP、记忆与规划](/zh-cn/stage-2/ag-002-tools-mcp-memory-and-planning/) — 解释智能体如何通过稳定工具契约、分层上下文和可更新记忆完成长任务。
3. [多智能体协作](/zh-cn/stage-2/ag-003-multi-agent-collaboration/) — 用角色、交接、共享状态、冲突处理和整体验收理解多智能体系统。
4. [AI Coding 与 Vibe Coding](/zh-cn/stage-2/ag-004-ai-coding-and-vibe-coding/) — 把需求、生成、运行、观察、修复、测试和审阅组织成可验收的开发循环。
5. [可观测性、评测与 Human-in-the-loop](/zh-cn/stage-2/ag-005-observability-evaluation-and-human-review/) — 从 trace、任务评测、权限和人工断点建立可运营、可恢复的智能体系统。

## 第 3 章 · [具身智能](/zh-cn/stage-3/)

从感知、决策、行动和反馈进入人形机器人、控制、强化学习、实训与群体具身智能。

1. [具身智能：感知、决策、行动与反馈](/zh-cn/stage-3/em-001-embodied-ai-loop/) — 建立具身智能的闭环模型，理解物理世界中的不确定性、实时性和安全约束。
2. [人形机器人与 Unitree 生态](/zh-cn/stage-3/em-002-humanoid-and-unitree/) — 从四足与人形机器人产品理解硬件、运动能力、应用形态和产业边界。
3. [多模态感知与环境理解](/zh-cn/stage-3/em-003-multimodal-perception-and-environment/) — 讲清视觉、深度、激光雷达和状态估计如何共同形成可供决策使用的环境表示。
4. [运动控制、强化学习与 Sim-to-Real](/zh-cn/stage-3/em-004-motion-control-rl-and-sim-to-real/) — 从状态、动作和奖励进入策略训练、仿真验证与真实机器人部署。
5. [G1、Go2 与 ROS2 实训](/zh-cn/stage-3/em-005-g1-go2-and-ros2-lab/) — 从安全检查、ROS2 节点和话题进入基础动作、状态读取与异常排查。
6. [Crowdverse 与群体具身智能](/zh-cn/stage-3/em-006-crowdverse-and-collective-embodied-ai/) — 用数字孪生训练场理解多智能体任务、群体行为、协同评测与仿真到现实。

## 第 4 章 · [行业应用](/zh-cn/stage-4/)

以落地方法为主线，覆盖教育、政务、企业、制造能源、交通、应急、海洋空间和文旅建筑。

1. [行业 AI 落地方法与 ROI](/zh-cn/stage-4/in-001-industry-ai-and-roi/) — 从业务问题、数据基础、系统集成、评测和投入产出判断行业 AI 是否值得落地。
2. [高校教学与科研](/zh-cn/stage-4/in-002-higher-education-and-research/) — 分析 AI 在备课、学习反馈、文献检索、科研写作和知识管理中的价值与边界。
3. [政务服务与乡村振兴](/zh-cn/stage-4/in-003-government-services-and-rural-revitalization/) — 用公共服务、基层治理和乡村振兴案例理解 AI 的流程价值、责任边界与可追溯要求。
4. [企业办公、知识管理与 AIGC](/zh-cn/stage-4/in-004-enterprise-office-knowledge-and-aigc/) — 把文档、会议、知识库和多模态内容生产连接成可复用、可审批的企业工作流。
5. [制造、能源与供应链](/zh-cn/stage-4/in-005-manufacturing-energy-and-supply-chain/) — 从工业数据、设备运行、生产质量、能源管理和供应链协同理解行业 AI 的闭环价值。
6. [交通与智慧机场](/zh-cn/stage-4/in-006-transportation-and-smart-airport/) — 用多模态交通大模型案例解释机场运行数据、知识服务、智慧运维和应急场景的系统集成。
7. [应急管理与火灾调查](/zh-cn/stage-4/in-007-emergency-and-fire-investigation/) — 以火灾调查辅助专家模型为例，理解证据链、法规知识、专业判断和责任边界。
8. [海洋与地理空间](/zh-cn/stage-4/in-008-ocean-and-geospatial-intelligence/) — 连接卫星遥感、浮标、船舶、时空模型和智能体工作流，形成可追溯的海洋空间服务。
9. [文旅、展厅与智慧建筑](/zh-cn/stage-4/in-009-tourism-exhibition-and-smart-building/) — 从导览机器人和数字工科大楼理解空间建模、路线、问答、数字孪生与长期运维。
