## PPT 生成 Markdown

`课程PPT框架/source/course-catalog.json` 是 27 节课程的权威映射。每节的 `slides.local_pptx` 指向本地同步源文件，`slides.url` 与 `slides.wiki_url` 指向同一份原始飞书课件。

维护时先在飞书精修课件，再同步本地 PPTX；随后依据 PPT 的知识顺序、案例、图片和视频修改对应 Markdown。网页可以保留独立媒体和原生 SVG 图解，但禁止发布整页幻灯片截图。

## 验证

```bash
pnpm courseware:verify
pnpm courseware:verify:remote
pnpm verify
```

这些检查分别核对本地 PPTX、飞书真实节点、Markdown 映射、媒体完整性和网站构建。公开权限仍是独立发布事项，不属于本次目录整理范围。
