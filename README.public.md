# 陈龙彪团队人工智能公开课

一套面向学习者的中文人工智能 Web 课程，共 4 章、27 节。每节课程由一篇 Markdown 讲义和一份本地 PPTX 组成；“幻灯片”按钮直接打开或下载该节课件。

- [打开课程网站](https://felixlark.github.io/ai-course-public/)
- [查看版权与素材说明](NOTICE.md)

## 课程结构

## 课件与讲义

课件从团队已有 PowerPoint 原稿中按课节选页，保留原始版式和可编辑结构。网页不内嵌、重建或模拟 PPT 播放器，也不把整页 PPT 或 PDF 渲染成图片。

## 本地运行与验证

```bash
pnpm install --frozen-lockfile
pnpm verify
pnpm dev -- --host 127.0.0.1
```

`pnpm verify` 会核对 27 个唯一 PPTX 映射、课件完整性、讲义结构、独立媒体、语言边界和 VitePress 构建。
