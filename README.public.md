# 陈龙彪团队人工智能公开课

一套面向学习者的中文人工智能 Web 课程，共 4 章、27 节。每节课程由一篇 Markdown 讲义和一份原始飞书课件组成：网站负责课程导航、图文讲解、独立图解和视频；“幻灯片”按钮直接打开飞书 Web PPT。

## 课件与讲义

飞书中的原始 PPTX 或原生 Slides 是课堂播放和精修的唯一课件。仓库不重新生成 PPT，也不实现、内嵌或模拟播放器。本地同步的 PPTX 只用于提取知识、图片和视频并校准 Markdown，不进入 GitHub Pages 发布包。

网页不会把整页 PPT 或 PDF 渲染成图片。课程只发布学习者讲义、经审查的独立图解和视频；每份公开媒体都应出现在解释相关知识的位置，而不是成为脱离正文的幻灯片图库。

## 本地运行与验证

```bash
pnpm install --frozen-lockfile
pnpm verify
pnpm dev -- --host 127.0.0.1
```

`pnpm verify` 会核对 27 个唯一课件映射、本地 PPTX 来源、讲义结构、独立媒体、语言边界和 VitePress 构建。需要同时核对飞书真实节点时运行：

```bash
pnpm courseware:verify:remote
```
